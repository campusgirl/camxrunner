What is this directory about?
_____________________________


This directory contains templates 
for the CAMxRunner:

	.
		contains templates to create scripts for HPC systems (so called external run)

	conf/ 
		contains general and model/version specific templates for configuration files
		These fies are used when a new configuration is built from scratch.
	
	modules/ 
		contains templates for modules
		Can be used by developers to extend CAMxRunner.
