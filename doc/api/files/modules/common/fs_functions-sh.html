common.fs.TryDecompressingFile

Checks if an input file is compressed, decompresses it and returns a new name.  Wo do this by searching files that have specific suffixes added to their name.  We decompress even if its a dry-run.

Due to potential permission issues, we decompress into temp files unless CXR_DECOMPRESS_IN_PLACE is true.  Therefore, we need to keep track which files where decompressed to which tempfiles.  This is stored in the hash CXR_GLOBAL_HASH_DECOMPRESSED_FILES.

If the decompression fails, we return the input string.

This function is directly used by common.runner.evaluateRule

Parameters

$1path to test
