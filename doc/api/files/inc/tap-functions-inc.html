ok

This is the workhorse method that actually prints the TAP compatible tests result.  This function is usually not called from outside (use okx,<pass> or fail instead) The output of this function goes to a TAP file and stdout.

Parameters

$1result (if this is 0, test ok, else nok)
$2name of the current test

is

Tests for string equality, the most important test function.

Example

is $(common.string.toLower HELLO) hello "common.string.toLower of HELLO is hello"

Parameters

$1actual result
$2expected result
[$3]test name

is.numerical

Tests for numerical equality.  Use this when comapring numbers, not is()

Example

is.numerical $(common.math.abs 0) 0 "common.math.abs of 0 is 0"

Parameters

$1actual result
$2expected result
[$3]test name
