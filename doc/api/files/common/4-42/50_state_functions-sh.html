store_state

Stores the current state of a run, can be either START, STOP or ERROR.  In the case of ERROR, processing stops immediately.

If a stage starts, checks if this stage was already executed.

We store the stage at which the process is currently, the caller must make sure To specify this as clearly as possible call it with “${CXR_TYPE_POSTPROCESS_SINGLE_DAY}@${CXR_META_MODULE_NAME}” (Choose proper TYPE of course!)

Returns

$CXR_RET_OK if OK $CXR_RET_ALREADY_RUN if this stage was already started

Parameters

$1State (START, STOP, ERROR)
$2Stage (CAMxRunner.sh, e. g.  ${CXR_TYPE_POSTPROCESS_SINGLE_DAY}@${CXR_META_MODULE_NAME})
