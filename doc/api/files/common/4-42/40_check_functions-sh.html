Estimates the requried space on the output filesystem.  Only a rough aproximation - hopefully an overestimation.

Here is what we might use in the future

CXR_DAYS CXR_OUTPUT_FREQUENCY CXR_NUMBER_OF_OUTPUT_SPECIES CXR_NUMBER_OF_GRIDS CXR_MASTER_GRID_COLUMNS CXR_MASTER_GRID_ROWS CXR_PROBING_TOOL
